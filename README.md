# 기리고 • Girigo PWA

> *"Dileğini yaz. Kabul edilirse bedeli ödenecek."*

If Wishes Could Kill (기리고) dizisinden ilham alınan eğlence amaçlı PWA uygulaması.

---

## 📱 Özellikler

- Dilek yazıp gönderme ekranı
- **5 dakika** sonra otomatik "Kabul Edildi" bildirimi
- **24 saatlik** geri sayım sayacı
- iOS ve Android'de tam ekran uygulama gibi çalışır
- Çevrimdışı destek (Service Worker)
- Kan damlası animasyonları ve korku estetiği

---

## 🚀 GitHub Pages'e Yükleme (Adım Adım)

### 1. GitHub'da yeni repo oluştur
- github.com → "New repository"
- İsim: `girigo` (veya istediğin isim)
- Public seç
- "Create repository"

### 2. Dosyaları yükle
```
index.html
manifest.json
sw.js
icon-192.png
icon-512.png
README.md
```

Tüm bu dosyaları repo'ya sürükle-bırak yap veya git ile push et.

### 3. GitHub Pages'i aç
- Repo → Settings → Pages
- Source: **Deploy from a branch**
- Branch: **main** → **/ (root)**
- Save

### 4. Birkaç dakika bekle
Link hazır:
```
https://KULLANICI_ADIN.github.io/REPO_ADIN/
```

---

## 📲 Telefona Kurulum

### Android (Chrome)
1. Linki Chrome ile aç
2. "Ana ekrana ekle" bildirimi çıkar → Kur
3. Uygulama ikonuyla açılır

### iOS (Safari)
1. Linki Safari ile aç
2. Paylaş butonu → "Ana Ekrana Ekle"
3. Uygulama ikonuyla açılır

---

## ⚙️ Gizli Sıfırlama

"Gönderildi" veya "Kabul Edildi" ekranındayken **logo'ya 3 kez hızla dokun** → sıfırlanır.
