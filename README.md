# 기리고 • Girigo PWA

> *"Dileğini yaz. Kabul edilirse bedeli ödenecek."*

If Wishes Could Kill (기리고) dizisinden ilham alınan eğlence amaçlı PWA uygulaması.

---

## 📱 Özellikler

- Dilek yazıp gönderme ekranı
- **5 dakika** sonra otomatik "Kabul Edildi" bildirimi
- **24 saatlik** geri sayım sayacı
- iOS ve Android'de tam ekran uygulama gibi çalışır
- Kan damlası animasyonları ve korku estetiği

---

## 📲 Telefona Kurulum

### Android (Chrome)
1. Linki Chrome ile aç
2. "Ana ekrana ekle" bildirimi çıkar → Kur
3. Uygulama ikonuyla açılır

### iOS (Safari)
1. Linki Safari ile aç
2. Paylaş butonu → "Ana Ekrana Ekle"
3. Uygulama ikonuyla açılır

---

## ⚙️ Gizli Sıfırlama

"Gönderildi" veya "Kabul Edildi" ekranındayken **logo'ya 3 kez hızla dokun** → sıfırlanır.

---

Bu uygulama tamamen eğlence amaçlıdır. If Wishes Could Kill (기리고) dizisinden ilham alınmıştır.
